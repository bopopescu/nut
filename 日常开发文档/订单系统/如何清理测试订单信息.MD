如何统一添加商品
    1.  建立 "果库复古大会" 用户
    2.  设置 该用户为 授权  卖家
    3.  用该用户添加商品
        ----
        后台判断, 如果添加者为 复古(seller manage add entity view,判断用户邮箱为 fugu@guoku.com)
        修改商品添加者为, 复古大会。 
        ----
    4.  如果商品已经在 库中, 把商品 ID 提交给 技术, 技术手动改动 添加者
        活动结束之后手动改回
    5.  商品管理员 用 复古大会 的账号登录, 管理商品
