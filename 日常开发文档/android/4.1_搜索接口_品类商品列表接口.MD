---------品类商品列表
mobile/v4/category/{{category_id}}/entity/

input : 
     offset :   页数
     count :   每页商品数
     reverse  :  缺省  0 ， 可以设为 1 ， 应该是 排序
      sort   : 缺省为 None ,  可以设为 'like'  按喜爱排序

response: 
     商品列表  格式 json 
     

---------用户搜索

mobile/v4/user/search/ 

input : q ，搜索词
           page,  页数 
           size， 每页的文章数量

response: 

res = {
            'stat' : {
                'all_count' : 数量,
            },
            'articles' : [  文章列表 ]
        }
