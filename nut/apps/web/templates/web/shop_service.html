选品服务计划

加入果库选品服务计划

「果库选品商店」服务计划，为活跃于线上的国内原创设计品牌与线下生活方式店们牵线。由果库提出与众不同的理想生活选品组合方案，去同质化的消费场景，营造更具情怀与体验感的复合式主题店，向消费者提供独特的购物体验，为线下零售业带去惊喜变化，同时为原创设计品牌扩展销售渠道。

目前，果库选品商店服务已经开启，合作商店列表，详见图文报道。希望各位能在逛店的同时，感受由果库营造的种种惊喜与不同以往的环境氛围。

如果你在打造自己的特色原创设计品牌，或者开设了追求生活品质的复合式实体店铺，都可以通过 bd@guoku.com 联系我们。


如何加入计划？


如果你是原创设计品牌，或者线上商家，

申请流程：

申请者发送申请邮件 → 果库审核资质 → 果库回复申请邮件 → 申请者与果库工作人员对接 → 选品确定及订货下单

申请邮件内容：

  • 1、邮件标题：<品牌名称> 申请加入果库选品服务计划；
  • 2、店铺介绍：名称、背景资料，以及必要的店址链接；
  • 3、设计师介绍：品牌主理人及设计师信息；
  • 4、商品介绍：申请加入贩售的商品清单，以及必要的商品信息、设计理念、材质说明等，愿闻其详；
  • 5、联系方式：QQ、微信号、手机号等。

审核标准：

  • 1、品牌符合果库精选商品之标准；
  • 2、品牌或某系列商品适合在现有线下店铺进行贩售；
  • 3、原创设计商品，不涉及抄袭、雷同、情色、暴力等问题。

如果你是线下店主，

申请流程：

申请者发送申请邮件 → 果库审核资质 → 果库回复申请邮件 → 申请者与果库工作人员对接 → 沟通选品需求及陈列面积等事宜 → 果库选品提案 → 沟通及确定 → 下单及进货 → 陈列完成开始售卖

申请邮件内容：

  • 1、邮件标题：<店名> 申请加入果库选品服务计划；
  • 2、店铺介绍：名称、地址、背景资料，含主营业务、风格介绍等；
  • 3、期望贩售商品之需求：风格、品类及品牌等；
  • 4、联系方式：QQ、微信号、手机号等。

审核标准：

  • 1、拥有营业执照等证件的合法商家；
  • 2、拥有独特的店铺风格，以及通过选品丰富消费体验的意愿。

最终解释权归果库所有。
